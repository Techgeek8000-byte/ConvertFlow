# ConvertFlow — Free Online Converters

Convert anything, instantly. 25+ file, data, and image converters — all processed in your browser.

🌐 **[convert-flow-beta.vercel.app](https://convert-flow-beta.vercel.app)**

## Features

- **25+ converters** — JSON ↔ CSV, XML ↔ JSON, YAML ↔ JSON, Markdown ↔ HTML, Image formats (PNG/JPEG/WebP/GIF/SVG), Base64 encode/decode, URL encode/decode, Hash generator (MD5/SHA), Color converter (HEX/RGB/HSL), and more
- **CSV ↔ JSON** — Parse CSV to structured JSON and vice versa with proper quoting
- **Image converter** — Convert between PNG, JPEG, WebP, GIF, SVG, and BMP in-browser
- **Developer tools** — JSON formatter, Regex tester, JWT decoder, Epoch converter
- **Free tier** — 15 conversions/day
- **Pro plans** — $1/week, $2/month, $12/year, or $25 lifetime
